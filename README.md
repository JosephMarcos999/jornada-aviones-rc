# Jornada de Aviones RC

Web informativa para una jornada de iniciación al mundo de los aviones RC tipo Cesna, donde los asistentes aprenderán los componentes básicos (motor brushless, ESC, batería LiPo, servos, emisora) y verán un montaje en directo.

Proyecto desarrollado con HTML5, Bootstrap 5 y CSS propio para la asignatura de Aplicaciones Web.
